# AgriSen 🌱

## Présentation

AgriSen est une plateforme web dédiée à la gestion et à la valorisation du secteur agricole. Elle permet aux utilisateurs d'accéder à des informations agricoles, de gérer des données liées aux exploitations et de faciliter les échanges autour des activités agricoles.

Le projet a été développé dans le cadre d'un projet académique afin de proposer une solution numérique moderne, simple d'utilisation et adaptée aux besoins du secteur agricole.

## Fonctionnalités

* Gestion des utilisateurs
* Gestion des exploitations agricoles
* Consultation des informations agricoles
* Interface web responsive
* Authentification sécurisée
* Tableau de bord de suivi

## Installation

### 1. Cloner le dépôt

```bash
git clone https://github.com/Agrisen-unchk/agrisen.git
```

### 2. Configurer la base de données

* Créer une base de données MySQL.
* Importer le fichier SQL du projet.
* Configurer les paramètres de connexion dans le fichier PHP concerné.

### 3. Lancer le projet

* Copier le projet dans le dossier `htdocs` (XAMPP) ou `www` (WAMP).
* Démarrer Apache et MySQL.
* Ouvrir :

```text
http://localhost/agrisen
```
